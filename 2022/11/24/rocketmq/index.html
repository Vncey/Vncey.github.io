    Mqbroker cannot start. No Mqbroker is running.

    Observation:
    rocketMq log Path: log/rocketmqlogs/

    broker.log

    broker_default.log

    store.log

    Solution

    1. delete abort file in the root/store folder
      • start the mqbroker again
      • if issue still persist , use step 2.
    2. delete root/store folder
      • restart rocketmq again , it will auto regenerate store folder.
      • start the mqbroker again.
      • if issue still persist, proceed to step 3.
    3. delete root/store folder, restart the server - sudo restart
      • start namesrv again
      • start mqbroker again
